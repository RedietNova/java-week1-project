# java-week1-project
##  Description
This is a simple Object-Oriented Programming (OOP) project written in Java.

The program simulates a person with different states such as energy, mood, and knowledge. The user can interact with the system through a menu and perform actions like studying, eating, and sleeping.

##  Features
- Create a person with a name
- Study to increase knowledge
- Eat to restore energy
- Sleep to recover energy and mood
- Check current status

##  Concepts Used
- Classes and Objects
- Constructors
- Methods
- Encapsulation
- User Input (Scanner)
- Loops and Conditional Statements

##  How to Run
1. Open the project in IntelliJ IDEA
2. Run the Main.java file
3. Enter your name
4. Choose actions from the menu
